# ATM Machine Simulation

A simple ATM simulation project written in C.

## Features:
- PIN verification
- Check balance
- Deposit money
- Withdraw money
- Mini statement
- Change PIN
- Exit

## Technologies Used:
- C Programming
- File Handling
- Conditional Statements
- Loops
- Switch Case

## How it works:
1. User enters PIN
2. Select ATM operations from menu
3. Perform transactions
4. View mini statement
5. Change PIN securely
